Founder

My first task at Mettle was to facilitate a rebrand.

I did this by creating an entirely new brand and testing this internally.

I began by creating a new colour palette and then created a new logo, logo mark and favicon.

I then compiled all of these assets into a branding guidelines deck for ease of use.

Following completion of the rebrand I was tasked with creating a reskin of the existing landing page utilising the new brand.

I also prototyped this design as can be seen below.

After completing the landing page I began to work on the dashboard interface.

Initially I began by creating a user flow diagram with the founder alongside completing an ideation session.

I then created a dashboard user interface using sketches provided by the founder.

I also created a prototype of the dashboard.

I am continuing to refine this dashboard with further iterations and will likely iterate again upon completion of usability testing.
